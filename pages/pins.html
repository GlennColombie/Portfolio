Project Spotlight

Disney Pins Monitor

Pins is a monitoring tool that tracks Disney Store EU pins, stores results in SQLite, and sends Discord alerts for new, restocked, sold-out, and price-change events.

Project Overview

Concept

Build a reliable watcher that checks collectible pin listings, detects meaningful changes, and reports them quickly in a clear format.

Core Features

  • Scrapes Disney Store EU pin listings with robust parsing.
  • Uses SQLite storage with image caching and schema migration.
  • Sends Discord webhook notifications for new pins, restocks, sold-out, and price changes.
  • Includes a protected Flask dashboard with health and summary endpoints.

Tech Stack

Python, Flask, SQLite, BeautifulSoup, requests/httpx, JavaScript, and CSS.

How It Runs

  • Reads configuration from environment variables and YAML.
  • Runs continuous scans (or single-run mode with --once).
  • Auto-starts dashboard in continuous mode and exposes health metrics.
  • Built with collector hype in mind, including quick tracking for fan-favorite drops like Nick Wilde themed items.
